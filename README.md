Smart Student Registration Portal

A modern and responsive multi-step Student Registration Portal developed using HTML, CSS, and JavaScript. The application streamlines the student enrollment process through interactive forms, real-time validation, fee calculation, academic performance analysis, password security checks, dark mode support, and local storage integration.

 ### Live Features

1.Multi-Step Registration Workflow

2.Real-Time Form Validation

3.Age Calculator Based on Date of Birth

4. Profile Photo Upload & Preview

5.Course Fee Calculator with GST & Discount

6.Academic Performance & Grade Calculator

7.Password Strength Checker

8.Dark Mode Support

9.Progress Tracking Bar

10. Registration Summary Preview

11.Responsive User Interface
